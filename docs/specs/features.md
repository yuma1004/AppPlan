# AppPlan 機能一覧 (Feature List)

AppPlanは、ローカルパス上の複数のプロジェクト（リポジトリ）にまたがる設計図やドキュメントを一元的に管理・編集できる「マルチプロジェクト設計IDE」デスクトップアプリケーションです。

以下に現在の主要な機能一覧をまとめます。

## 1. マルチプロジェクト一元管理 (Multi-Project Management)
- **複数プロジェクトの同時展開**: PC内の複数のディレクトリをサイドバーに追加し、IDE空間内で一元管理・切り替えが可能。
- **プロジェクト状態の永続化**: 追加したプロジェクトのパスや「最近開いたプロジェクト履歴（最大10件）」は Local Storage に自動保存され、次回起動時も状態を復元。
- **設計図の自動検出**: プロジェクトルート内の `docs/` 配下にある特定ディレクトリ（`appplan`, `diagrams`, `architecture`, `specs`, `design` 等）から自動的にMarkdownファイルをスキャンしてリスト化。
- **サンプル作成**: 使い方をすぐに理解するためのダミーデータの入った「サンプルワークスペース」をワンクリックで生成する機能。

## 2. インテリジェント・エディタ (Mermaid Diagram Editor)
- **ライブオートセーブ**: コードを編集すると 500ms のデバウンスで即座にローカルのファイルシステム（Markdownファイル）へ自動保存。
- **YAMLフロントマター完全対応**: 各Markdownのヘッダー情報（タイトル、説明、図のタイプなど）を自動解析し、UIのタイトルバー（WorkspaceHeader）上でリッチに表示。
- **図のファイル管理**: UI上からの直接的な「新規ダイアグラム作成」および「ファイルの削除」機能。
- **レイアウトの柔軟な切り替え**: エディタとプレビュー画面の配置を「横並び（Split Right）」と「縦並び（Split Bottom）」でワンタッチで切り替え。
- **ペインのリサイズ**: `react-resizable-panels` による直感的なドラッグ＆ドロップでの画面幅・高さの調整。

## 3. 高機能プレビューパネル (Advanced Preview Panel)
- **Mermaid.js のリアルタイム描画**: ダークテーマのIDEに最適化された配色のMermaidレンダリングを即座に実行。
- **シームレスなズーム機能**: `Ctrl + マウスホイール` を使用して、プレビュー空間内の図を 10% 〜 500% までシームレスに拡大・縮小。他の図へ切り替えた際は自動で倍率をリセット。
- **完全フルスクリーン展開**: ポップアップダイアログではなく、Z-Indexを制御したネイティブな全画面表示モードを搭載。`Esc` キーで素早く通常画面へ復帰。
- **ワンタッチ翻訳エンジン**: 英語で記述されたER図などのノード名（ラベル）を、Mermaidの構文構造を壊すことなく安全に日本語へ置換する翻訳トグル機能。
- **高画質スクリーンショット保存**: レンダリングされた図を、OSの画像処理レイヤーを経由して高解像度のままPNG画像としてローカルディレクトリにワンクリックで出力（保存）。
- **エラーフェイルセーフ**: Mermaidの構文エラー発生時は描画をクラッシュさせず、エラー箇所を安全にパネル内にテキスト出力。

## 4. Git ステータス連動可視化 (Git Integration)
- **リポジトリごとの差分抽出**: 登録したプロジェクトのパスがGitリポジトリ（`--is-inside-work-tree`）である場合、裏側で `git status --porcelain` を実行し状態を抽出。
- **ファイルレベルのバッジ可視化**: サイドバー上で図を展開した際、ファイル単位のステータス（M: 修正, A: 追加, U: 未追跡, D: 削除 など）を色付きのバッジで可視化。
- **プロジェクトレベルの集計表示**: 変更があるプロジェクトには、GitHubアイコンとともに未コミットの変更ファイル総数をリアルタイムに表示。
- **Git Only モード（差分フィルター）**: サイドバーのフィルターアイコン（漏斗）をクリックすることで、**「未コミットの変更があるプロジェクトのみ」**を抽出し、複数プロジェクトを横断した作業時のノイズを完全に除去。
- **完全なリアルタイム同期**: 図の編集・新規作成・保存時に自動で非同期の再スキャンが走り、ターミナル等の外部操作なしにバッジの状態が常に最新と連動。

## 5. 洗練された UI/UX デザイン
- **64px 統一ヘッダー**: 左サイドバー、エディタ、プレビューパネルの全ての上部ヘッダー高さを正確に `64px` に統一。画面を横断するソリッドでプロフェッショナルな境界線を構築。
- **ノンブロッキング・ツールバー**: プレビューヘッダーにネイティブにツールバーを埋め込むことで、キャンバス上の図に浮遊ボタンが被ってしまう問題を排除。フルスクリーン時には自動で美しい半透明のフローティングバーへとトランスフォーム。
- **スマートな領域確保**: 各パネルのスクロール領域や余白（Padding）を限界まで最適化し、長大なアーキテクチャ設計図でも全体を見渡しやすい広いワーキングエリアを提供。
